运行简单的本地服务器

准备工作：html文件

步骤：

  1. 安装python,可通过在命令行窗口执行 python -V命令查看是否安装成功
  2. 使用cd 命令切换到html文件所在的目录
  3. 执行python -m http.server命令运行http服务器(默认端口为8000，若端口被占用，可通过python -m http.server 7000命令指定端口号7000，端口号可自定义修改)
  4. 在浏览器中输入localhost:8000访问本地网站(若端口号已自定义修改，此处也随之修改)，亲测可用
